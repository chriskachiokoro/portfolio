Private Client Project

AI JPEG Interrogation

Improved extraction accuracy from ~50% to 85–95% on standard manufacturing drawings.

Overview

Built and improved an AI pipeline that converts manufacturing JPEG drawings into structured JSON fields while protecting client-sensitive data.

Problem

Manual data entry from drawings was slow, error-prone, and struggled to extract detailed text blocks from varied scan qualities.

Workflow

JPEG → orientation correction → notes crop → vision model extraction → priority-based merge → Python validation → JSON/CSV outputs.

My Contribution

  • Root-cause analysis and testing
  • Prompt refinement and evaluation
  • Python post-processing logic
  • Docker setup and validation

Results

Accuracy increased from roughly 50% to 85–95% on standard drawings. Confidential source files and code are not public.

Tech Stack

GPT Vision, Python, Docker, OCR, JSON
